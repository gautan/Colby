Recommendations by Use Case

Best for AWS + EKS Native Deployment

Winner: Apache APISIX

  • Native Kubernetes CRDs and Ingress Controller
  • Excellent Helm chart with production-ready defaults
  • Hot reload without downtime via etcd/Admin API
  • 80+ built-in plugins covering all requirements

Runner-up: Kong Gateway (more mature enterprise ecosystem, higher cost for enterprise features)

Best for Enterprise Auth & Entitlements

Winner: Spring Cloud Gateway

  • Native Spring Security with OIDC/SAML support
  • Deep Java ecosystem integration (familiar to enterprise teams)
  • Excellent for complex authorization logic
  • Best choice if team has strong Java/Spring expertise

Runner-up: Kong Enterprise (dedicated OIDC plugin, 24/7 support)

Best for High Throughput / Low Latency

Winner: Apache APISIX

  • NGINX + LuaJIT core delivers sub-millisecond latency overhead
  • Benchmarks show 10x+ performance vs Java-based gateways
  • Excellent connection pooling and HTTP/2/3 support

Runner-up: Envoy Gateway (proven at Google/Lyft scale, slightly higher complexity)

Best for Lowest Operational Risk

Winner: Kong Enterprise

  • 24/7 enterprise support with SLAs
  • Most extensive documentation and training resources
  • Proven track record at Fortune 500 companies
  • Enterprise dashboard with RBAC

Runner-up: AWS API Gateway (fully managed, zero infrastructure, but limited customization)

Homebrew vs Off-the-Shelf Analysis

NGINX + Node.js Custom Gateway: NOT RECOMMENDED

Building a custom gateway requires significant ongoing investment with limited return.

Factor Homebrew (NGINX+Node) Off-the-Shelf (APISIX/Kong)
Initial Development Time 6-12 months 2-4 weeks
Feature Completeness ~40% of requirements ~95% of requirements
Ongoing Maintenance 2-3 FTEs dedicated 0.25-0.5 FTE
Security Patches Self-managed (risk) Community/Vendor provided
Feature Velocity Slow (internal roadmap) Fast (active community)
Knowledge Transfer High risk (tribal knowledge) Low risk (industry standard)
Hiring/Training Custom training required Industry skills transferable
Total 3-Year TCO $600K-$1M+ $50K-$200K

When Homebrew MIGHT Make Sense

  • API Gateway is your core product (not infrastructure)
  • Extreme edge cases not covered by any existing solution
  • Regulatory requirements prohibiting third-party code
  • Unlimited engineering budget and timeline
  • Willingness to maintain for 10+ years

Operational Risks of Homebrew

  • Security debt: CVEs require custom patching
  • Bus factor: Original developers leave, knowledge is lost
  • Feature lag: Always catching up to industry standards
  • Testing burden: Must validate every edge case yourself
  • Compliance: Self-attestation for SOC2/PCI audits
